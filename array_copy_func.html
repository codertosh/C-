C# Array Copy Methods

1. Array.Copy()

int[] dest = new int[source.Length];
Array.Copy(source, dest, source.Length);
✓ Modifies existing array ✓ Fast for large arrays ✗ Requires pre-initialized array

2. Clone()

int[] clone = (int[])source.Clone();
✓ Creates new array ✗ 1D arrays only ✗ Returns object

3. CopyTo()

int[] dest = new int[source.Length];
source.CopyTo(dest, 0);
✓ Instance method ✓ Specify start index ✗ Needs initialized array

4. LINQ ToArray()

using System.Linq;
int[] copy = source.ToArray();
✓ Concise syntax ✗ LINQ overhead ✓ Returns new array

5. Deep Copy

// For reference types
Person[] deepCopy = people
    .Select(p => new Person { Name = p.Name })
    .ToArray();
✓ Copies objects ✗ Manual implementation ✓ Isolated copies

When to Use:

• Value types → Any shallow copy
• Reference types → Deep copy
• Simple needs → Clone()/ToArray()
• Large data → Array.Copy
