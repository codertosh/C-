Checked Arrays in C# - Quick Notes

1. What is a Checked Context?

  • In C#, the checked keyword detects integer overflow
  • Not array-specific but crucial for array calculations

2. Syntax of checked in Arrays

int[] numbers = { int.MaxValue, 1 }; try { int result = checked(numbers[0] + numbers[1]); // Throws OverflowException Console.WriteLine(result); } catch (OverflowException ex) { Console.WriteLine("Overflow detected: " + ex.Message); }

3. Using unchecked to Allow Overflow

int result = unchecked(int.MaxValue + 1); // Wraps to negative value

4. Checked Block for Multiple Operations

checked { int sum = numbers[0] + numbers[1]; // Throws exception if overflow Console.WriteLine(sum); }

5. Bounds Checking in Arrays

int[] arr = { 1, 2, 3 }; Console.WriteLine(arr[5]); // ‚ùå IndexOutOfRangeException

C# automatically checks array bounds!

6. When to Use Checked Context?

  • Large number operations in arrays
  • Financial/scientific calculations
  • Prevent silent overflow errors
