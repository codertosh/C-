Jagged Arrays in C#

What is a Jagged Array?

A jagged array in C# is an array of arrays, meaning each row can have a different number of elements.

Declaring a Jagged Array

int[][] jaggedArray = new int[3][]; // 3 rows with undefined columns

Initializing a Jagged Array

jaggedArray[0] = new int[] { 1, 2, 3 };
jaggedArray[1] = new int[] { 4, 5 };
jaggedArray[2] = new int[] { 6, 7, 8, 9 };

Accessing Elements

Console.WriteLine(jaggedArray[0][1]); // Output: 2
Console.WriteLine(jaggedArray[2][3]); // Output: 9

Looping Through

for Loop:

// #1

for (int i = 0; i < jaggedArray.Length; i++) {
  

// #2

for (int j = 0; j < jaggedArray[i].Length; j++) {
    Console.Write(jaggedArray[i][j] + " ");
  }

//#1. jaggedArray is an array of arrays, so Length returns the number of rows (or sub-arrays) in the jagged array.

//#2. jaggedArray[i] is a single array (row), Length returns the number of elements in that row.

}

  • Outer loop (i) runs from 0 to jaggedArray.Length - 1 (number of rows).

  • Inner loop (j) runs from 0 to jaggedArray[i].Length - 1 (number of elements in the current row).

foreach Loop:

foreach (int[] row in jaggedArray) {
  foreach (int num in row) {
    Console.Write(num + " ");
  }
}

Comparison

Feature Jagged Array Multidimensional Array
Memory Usage More efficient Less efficient
Row Sizes Variable Fixed
Syntax arr[0][1] arr[0,1]

Multidimensional Array Example

int[,] multiArray = { { 1, 2 }, { 3, 4 }, { 5, 6 } };
Console.WriteLine(multiArray[1, 0]); // Output: 3

When to Use Jagged Arrays

✓ When rows have varying lengths
✓ When memory efficiency is important
✓ For irregular datasets
